# text_system
【文本情感分析识别】Python+SVM分类算法+情感分析+机器学习

## 一、介绍
使用Python作为开发语言，基于文本数据集（一个积极的xls文本格式和一个消极的xls文本格式文件），使用Word2vec对文本进行处理。通过支持向量机SVM算法训练情绪分类模型。实现对文本消极情感和文本积极情感的识别。并基于Django框架开发网页平台实现对用户的可视化操作和数据存储。
